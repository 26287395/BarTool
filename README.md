# BarTool

A clean and beautiful macOS menu bar app manager.

Features:

- View running applications
- Search apps
- CPU / Memory monitoring
- Quick app termination
- Double-click to activate app
- Launch at login
- Hide system processes
- Hide Apple apps
- Native menu bar window mode

---

- Retina icon support
